
## Description

Billing System app - implemented with nodejs (nestjs) and React (material-ui)

## Installation

```bash
$ npm install && npm run install:client && npm run build:client
```

## Running the app

```bash
* The app is served on 8080

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
